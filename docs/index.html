工作台处理流程，PARA 自动归位

快记内容落入工作台的「待分拣」区。确认分类后进入「待提炼」区。改一个 PARA 标签，内容自动归入对应区域——不需要建文件夹。

  • ①工作台待分拣 / 待提炼双区块
  • ②PARA 四区域自由切换
  • ③改标签即完成归类
  • ④侧边栏拖拽调整宽度，收起时 hover 浮窗

从原子笔记到项目，知识层层沉淀

每篇笔记都是独立的原子知识块。标注重点、写纲要总结后，可关联到「行动」中的项目。项目完结时经验不会丢失——下一篇项目笔记里，相关素材已经就绪。

  • ①标注重点，写纲要——渐进式归纳
  • ②属性面板：标签 / 分类 / 项目 / 提炼状态
  • ③原子笔记关联到「行动」中的项目
  • ④项目笔记中调用已有原子笔记，不零启动

行动工作台，项目统筹

行动是独立的项目工作区。从 PARA 各区域调用原子笔记，组织成项目看板。项目做完经验留存，下一个项目直接复用已有素材。

  • ①创建项目，关联原子笔记
  • ②项目看板统筹所有素材
  • ③项目完结，经验自动归档
  • ④新项目启动直接调用已有素材

所有笔记统筹，全局检索

表格视图显示所有原子笔记，按来源、分类、项目、标签、提炼状态多维度筛选排序。全文搜索毫秒级响应，跨 PARA 区域统一检索。

  • ①表格视图：按多维度筛选排序
  • ②全文搜索毫秒级响应
  • ③状态标签追踪提炼进度
  • ④跨 PARA 区域统一检索

Tiptap 引擎驱动，16 种块类型

不只是写笔记。代码高亮、表格、折叠块、待办清单、笔记嵌入与 @ 引用——每一个功能都为深度思考和知识连接设计。

Slash 命令菜单

输入 / 呼出块插入菜单，段落/标题/代码/表格/折叠/图片快速切换。

浮动格式工具栏

选中文字弹出气泡菜单，加粗/斜体/链接/高亮，不打断写作心流。

笔记嵌入 & 引用

在笔记中嵌入其他笔记，或 @ 引用——知识自然链接成网络。

块级拖拽重排

拖拽手柄调整段落顺序，重组内容像乐高一样直观。

浏览器式多标签

同时打开多篇笔记，前进/后退历史栈——像浏览器一样切换上下文。

属性面板

纲要/来源/标签/分类/项目/提炼状态——元数据驱动，搜索摘要可见。
